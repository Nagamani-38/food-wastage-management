# Food Wastage Management System

## Project Overview
This project helps reduce food wastage by connecting food providers with receivers.

## Tech Stack
- Python
- Streamlit
- SQL
- Power BI
- Pandas
- Matplotlib

## Modules
- Providers
- Receivers
- Food Listings
- Claims
- Analysis Dashboard

## Features
- Provider registration
- Receiver registration
- Food listing management
- Claims management
- Data analysis dashboard

## Deployment
Streamlit Cloud
